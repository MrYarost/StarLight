📜 Правила игры

1. Правила для игры на официальных серверах

  • Использование читов запрещено. Это легко может заметить анти-чит или администрация (если игроки пожалуются и скинут доказательства) — бан по IP на официальных серверах.
  • Использование багов для засорения сервера или использования против настоящих игроков (жёсткие бусты) — бан по IP на официальных серверах.
  • Игра с одного IP более 3 сеансов — кик со всех устройств.

📜 Правила сообщества

Обязательно прочитайте перед публикацией своей карты.

1. Общие положения

Запрещено спамить несколько одинаковых постов — за это вы будете забанены на официальных серверах и удаление всех постов.

2. Правила для публикации

  • Если на вашей карте будет что-то связанное с темами 18+ (убийства, кровь, что-то связанное с сексом) — бан по IP на официальных серверах.
  • Если в описании или названии будет написана всякая дичь, или что-то из пункта 1, или указан недействительный IP — удаление поста.
  • Публикация карт, которые не настроены — удаление карты.

Ответственность несёте вы!

⚠️ Важно: Нарушение правил может привести к блокировке вашего поста, аккаунта, сервера, карты и вашего IP.
